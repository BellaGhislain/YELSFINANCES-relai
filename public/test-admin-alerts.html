Instructions de test

  1. Alertes de base : Cliquez sur les boutons pour afficher différents types d'alertes. Elles se ferment automatiquement après 5 secondes.
  2. Toasts : Les notifications toast apparaissent en haut à droite et se ferment automatiquement après 3 secondes.
